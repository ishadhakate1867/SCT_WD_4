# Advanced To-Do Web App

A modern and responsive To-Do List application built using HTML, CSS, and JavaScript. The application helps users organize daily tasks with features such as task management, priorities, categories, search, filtering, dark mode, and local storage.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed or pending
- Set due date
- Set due time
- Task priority (High, Medium, Low)
- Task categories
- Search tasks
- Filter tasks (All, Completed, Pending)
- Favorite important tasks
- Progress bar
- Task statistics
- Dark/Light mode
- Local Storage support
- Clear completed tasks
- Clear all tasks
- Fully responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome
- Google Fonts (Poppins)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/advanced-todo-app.git
```

### 2. Open the project folder

```bash
cd advanced-todo-app
```

### 3. Run the application

Open **index.html** in your preferred web browser.

No additional installation or dependencies are required.

## How It Works

1. Enter a task title.
2. Add a description (optional).
3. Select a due date and time.
4. Choose a priority level.
5. Select a category.
6. Click **Add Task**.
7. Manage tasks using Complete, Edit, Favorite, and Delete buttons.
8. Use Search and Filters to organize tasks.
9. Switch between Light and Dark mode.
10. Tasks are automatically saved using Local Storage.
